# Skill: DataGenerationVisualizer

**Purpose:** Renders the complex JSON structure of a QA interaction into an easily readable and visually appealing HTML view for manual triage.

**Core Capabilities:**
1. Loads output JSON models and structures them via semantic HTML logic.
2. Formats CoT elements inside distinct collapsable or visual boxes.
3. Automatically triggers browser rendering upon success if configured to do so.
